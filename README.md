My dotfiles for:

- Alacritty

## Example

![alt example](https://github.com/paulkoeckdev/DotFiles/blob/main/example.png?raw=true)

### Installation

```bash
git clone https://github.com/paulkoeckdev/dotfiles ~/dotfiles
cd ~/dotfiles
chmod +x ./install.sh
./install.sh
```
